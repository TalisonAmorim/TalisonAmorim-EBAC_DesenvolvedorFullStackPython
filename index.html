Sobre

Este é um projeto desenvolvido como parte do curso de Desenvolvedor Full Stack Python da EBAC.

O objetivo é criar um formulário de cadastro utilizando HTML, CSS, JavaScript e jQuery, com os campos: nome completo, e-mail, telefone, CPF, endereço completo e CEP, utilizando o plugin jQuery Mask Plugin para aplicar máscara aos campos CPF, telefone e CEP
